Shuleni - Frequently Asked Questions

What is Shuleni and how does it help schools manage report cards?

Shuleni is an offline report card management system built for schools that want to simplify how they record, process, and print student results. Instead of using Excel or manual entry books, teachers can enter marks directly into the Shuleni offline Desktop application, calculate averages automatically, and generate professional PDF report cards in seconds.

Shuleni helps secondary school and primary schools in Uganda to reduce paperwork, save time, and ensure accuracy in report card preparation. Every report card includes grades, descriptors, generic skills, AOI scores, and Project scores.

How does Shuleni work for teachers and school administrators?

Shuleni allows teachers to create classes, add students, input marks, and generate report cards instantly. School administrators (like headteachers or deputies) can review all entries, set academic terms, set custom comments, and download compiled results for record keeping.

The system works through a Windows desktop application that can be accessed on school and personal computers. Data is stored securely, and you can also import or export from Excel or CSV files, making Shuleni easy to integrate with existing school records.

How do schools generate report cards with Shuleni Report Card Management System?

Report card generation with Shuleni software is very easy.

1. Register student names.
2. Enter student scores.
3. Go to settings and set number of AOIs, subjects, Term's End & Begining dates.
4. Upload school badge.
5. Click Print Report Cards.
6. Report cards will be output in PDF format.

How can I move student records between PCs?

Shuleni's import & export features allow the teacher/headteacher to export students assessment records to Excel/CSV. The exported records can then be imported into Shuleni on any other computer.
To import/export:

1. Click settings.
2. Navigate to the Import/Export feature.
3. Select the class to Import or Expport.
4. Click on either Import or Export.
5. Pop-up will notify you that records have been exported/imported.

Can I use Shuleni Report Card Management System without an internet connection?

Absolutely. Shuleni is 100% offline. Once you have downloaded and installed the Desktop application on your computer, it runs offline. In case you want to switch computers or share records, you can export to Excel/CSV and share the records as a CSV/Excel file. The receipient imports the file into Shuleni on their computer.
